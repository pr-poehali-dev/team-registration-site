🤖 НАСТРОЙКА TELEGRAM БОТА НА TIMEWEB ХОСТИНГЕ

═══════════════════════════════════════════════════════════

ШАГ 1: СОЗДАТЬ БОТА В TELEGRAM
───────────────────────────────

1. Открой Telegram и найди @BotFather
2. Отправь команду: /newbot
3. Придумай имя бота (например: "Tournament Registration Bot")
4. Придумай username для бота (например: "ce876244_tournament_bot")
5. Скопируй токен бота (выглядит так: 123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ)

═══════════════════════════════════════════════════════════

ШАГ 2: НАСТРОИТЬ ФАЙЛЫ
───────────────────────

1. Открой файл: telegram-bot.php
2. Найди строку: $bot_token = 'ВАШ_ТОКЕН_БОТА';
3. Замени на: $bot_token = '123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ';
   (вставь свой токен)

4. Открой файл: setup-telegram-webhook.php
5. Найди строку: $bot_token = 'ВАШ_ТОКЕН_БОТА';
6. Замени на тот же токен

═══════════════════════════════════════════════════════════

ШАГ 3: ЗАГРУЗИТЬ НА ХОСТИНГ
────────────────────────────

1. Скачай билд: Скачать → Скачать билд
2. Загрузи ВСЕ файлы на Timeweb через File Manager
3. Убедись что файлы загружены:
   ✓ telegram-bot.php
   ✓ setup-telegram-webhook.php

═══════════════════════════════════════════════════════════

ШАГ 4: УСТАНОВИТЬ WEBHOOK
──────────────────────────

1. Открой в браузере:
   https://ce876244.tw1.ru/setup-telegram-webhook.php

2. Должна появиться зелёная галочка:
   ✅ Webhook успешно установлен!

3. Если ошибка - проверь токен бота в файлах

═══════════════════════════════════════════════════════════

ШАГ 5: ПРОВЕРИТЬ БОТА
──────────────────────

1. Найди своего бота в Telegram (по username)
2. Нажми START или отправь: /start
3. Бот должен ответить приветственным сообщением

Команды бота:
• /start - Приветствие
• /register - Регистрация команды
• /myteam - Информация о команде
• /cancel - Отменить регистрацию
• /help - Помощь

═══════════════════════════════════════════════════════════

ШАГ 6: БЕЗОПАСНОСТЬ
────────────────────

УДАЛИ файл setup-telegram-webhook.php после настройки!

Через File Manager на Timeweb или через FTP.

═══════════════════════════════════════════════════════════

ГОТОВО! 🚀

Бот работает и принимает команды!

Пользователи могут:
1. Писать боту /register
2. Переходить на сайт для регистрации
3. Проверять статус через /myteam
